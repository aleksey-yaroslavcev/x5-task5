1. создать CRUD интерфейс CustomList с методами
	void add(Object);		добавить элемент
	void add(Object, index); 	добавить элемент в определенное место
	object get(index)		получить элемент
	Object update (Object,index)	возврщает объект который был в коллекции по указанному индексу и заменяет его новым
	Object delete (Object)		удалить элемент, возврщает объект который был в коллекции
	int size()	возвращает колличество элементов
